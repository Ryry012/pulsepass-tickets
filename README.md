# PulsePass Tickets

A modern ticket marketplace website built with Next.js 15, React, TypeScript, TailwindCSS, Framer Motion, and lucide-react.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Build

```bash
npm run build
```

## Included

- Responsive event marketplace homepage
- Search input and category filtering
- Trending event cards with local UI state
- Selected-event reservation panel
- Seller feature blocks
- Newsletter signup form shell
- SEO metadata and OpenGraph basics
